## 题目描述

给你一个二叉搜索树的根节点 root ，返回 树中任意两不同节点值之间的最小差值 。

示例 1：
<br>输入：root = [4,2,6,1,3]
<br>输出：1

示例 2：
<br>输入：root = [1,0,48,null,null,12,49]
<br>输出：1

提示：
<br>树中节点数目在范围 [2, 100] 内
<br>0 <= Node.val <= 105
