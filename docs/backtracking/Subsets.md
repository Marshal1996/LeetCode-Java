## 题目描述

给你一个整数数组 nums ，数组中的元素 互不相同 。返回该数组所有可能的子集（幂集）。

解集 不能 包含重复的子集。你可以按 任意顺序 返回解集。

 

示例 1：
<br>输入：nums = [1,2,3]
<br>输出：[[],[1],[2],[1,2],[3],[1,3],[2,3],[1,2,3]]

示例 2：
<br>输入：nums = [0]
<br>输出：[[],[0]]

提示：
<br>1 <= nums.length <= 10
<br>-10 <= nums[i] <= 10
<br>nums 中的所有元素 互不相同
