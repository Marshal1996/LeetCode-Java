## 题目描述

给你一个链表的头节点 head 和一个整数 val ，请你删除链表中所有满足 Node.val == val 的节点，并返回 新的头节点 。

示例 1：
<br>输入：head = [1,2,6,3,4,5,6], val = 6
<br>输出：[1,2,3,4,5]

示例 2：
<br>输入：head = [], val = 1
<br>输出：[]

示例 3：
<br>输入：head = [7,7,7,7], val = 7
<br>输出：[]

提示：
<br>列表中的节点在范围 [0, 104] 内
<br>1 <= Node.val <= 50
<br>0 <= k <= 50
