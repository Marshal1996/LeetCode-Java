## 题目描述

给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是回文。 给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

 

示例 1：
<br>输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
<br>输出：6
<br>解释：连续子数组 [4,-1,2,1] 的和最大，为 6 。

示例 2：
<br>输入：nums = [1]
<br>输出：1

示例 3：
<br>输入：nums = [0]
<br>输出：0

示例 4：
<br>输入：nums = [-1]
<br>输出：-1

示例 5：
<br>输入：nums = [-100000]
<br>输出：-100000

提示：
<br>1 <= nums.length <= 3 * 104
<br>-105 <= nums[i] <= 105
