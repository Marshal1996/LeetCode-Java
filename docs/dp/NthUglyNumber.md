## 题目描述

给你一个整数 n ，请你找出并返回第 n 个 丑数 。

丑数 就是只包含质因数 2、3 和/或 5 的正整数。

 

示例 1：
<br>输入：n = 10
<br>输出：12
<br>解释：[1, 2, 3, 4, 5, 6, 8, 9, 10, 12] 是由前 10 个丑数组成的序列。

示例 2：
<br>输入：n = 1
<br>输出：1
<br>解释：1 通常被视为丑数。

提示：
<br>1 <= n <= 1690
