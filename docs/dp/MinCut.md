## 题目描述

给你一个字符串 s，请你将 s 分割成一些子串，使每个子串都是回文。

返回符合要求的 最少分割次数 。

 

示例 1：
<br>输入：s = "aab"
<br>输出：1
<br>解释：只需一次分割就可将 s 分割成 ["aa","b"] 这样两个回文子串。

示例 2：
<br>输入：s = "a"
<br>输出：0

示例 3：
<br>输入：s = "ab"
<br>输出：1

提示：
<br>1 <= s.length <= 2000
<br>s 仅由小写英文字母组成
