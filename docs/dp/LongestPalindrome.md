## 题目描述

给你一个字符串 s，找到 s 中最长的回文子串。

 

示例 1：
<br>输入：s = "babad"
<br>输出："bab"
<br>解释："aba" 同样是符合题意的答案。

示例 2：
<br>输入：s = "cbbd"
<br>输出："bb"

示例 3：
<br>输入：s = "a"
<br>输出："a"

示例 4：
<br>输入：s = "ac"
<br>输出："a"

提示：
<br>1 <= s.length <= 1000
<br>s 仅由数字和英文字母（大写和/或小写）组成
