## 题目描述

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

 

示例 1：
<br>输入：height = [0,1,0,2,1,0,1,3,2,1,2,1]
<br>输出：6
<br>解释：上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

示例 2：
<br>输入：height = [4,2,0,3,2,5]
<br>输出：9

提示：
<br>n == height.length
<br>0 <= n <= 3 * 104
<br>0 <= height[i] <= 105
