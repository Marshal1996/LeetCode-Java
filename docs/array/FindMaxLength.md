## 题目描述

给定一个二进制数组 nums , 找到含有相同数量的 0 和 1 的最长连续子数组，并返回该子数组的长度。

 

示例 1:
<br>输入: nums = [0,1]
<br>输出: 2
<br>说明: [0, 1] 是具有相同数量0和1的最长连续子数组。

示例 2:
<br>输入: nums = [0,1,0]
<br>输出: 2
<br>说明: [0, 1] (或 [1, 0]) 是具有相同数量0和1的最长连续子数组。

提示：
<br>1 <= nums.length <= 105
<br>nums[i] 不是 0 就是 1
