## 题目描述

如果数组是单调递增或单调递减的，那么它是单调的。

如果对于所有 i <= j，A[i] <= A[j]，那么数组 A 是单调递增的。 如果对于所有 i <= j，A[i]> = A[j]，那么数组 A 是单调递减的。

当给定的数组 A 是单调数组时返回 true，否则返回 false。

 

示例 1：
<br>输入：[1,2,2,3]
<br>输出：true

示例 2：
<br>输入：[6,5,4,4]
<br>输出：true

示例 3：
<br>输入：[1,1,1]
<br>输出：true

提示：
<br>1 <= A.length <= 50000
<br>-100000 <= A[i] <= 100000
