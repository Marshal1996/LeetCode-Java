## 题目描述

给定一组非负整数 nums，重新排列每个数的顺序（每个数不可拆分）使之组成一个最大的整数。

注意：输出结果可能非常大，所以你需要返回一个字符串而不是整数。

示例 1：
<br>输入：nums = [10,2]
<br>输出："210"

示例 2：
<br>输入：nums = [3,30,34,5,9]
<br>输出："9534330"

示例 3：
<br>输入：nums = [1]
<br>输出："1"

示例 4：
<br>输入：nums = [10]
<br>输出："10"

提示：
<br>1 <= nums.length <= 100
<br>0 <= nums[i] <= 109
