## 题目描述

给定一个非负整数 c ，你要判断是否存在两个整数 a 和 b，使得 a2 + b2 = c 。

 

示例 1：
<br>输入：c = 5
<br>输出：true
<br>解释：1 * 1 + 2 * 2 = 5

示例 2：
<br>输入：c = 3
<br>输出：false

示例 3：
<br>输入：c = 4
<br>输出：true

示例 4：
<br>输入：c = 2
<br>输出：true

示例 5：
<br>输入：c = 1
<br>输出：true

提示：
<br>0 <= c <= 231 - 1
