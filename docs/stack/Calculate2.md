## 题目描述

给你一个字符串表达式 s ，请你实现一个基本计算器来计算并返回它的值。

整数除法仅保留整数部分。

 

示例 1：
<br>输入：s = "3+2*2"
<br>输出：7

示例 2：
<br>输入：s = " 3/2 "
<br>输出：1

示例 3：
<br>输入：s = " 3+5 / 2 "
<br>输出：5

提示：
<br>1 <= s.length <= 3 * 105
<br>s 由整数和算符 ('+', '-', '*', '/') 组成，中间由一些空格隔开
<br>s 表示一个 有效表达式
<br>表达式中的所有整数都是非负整数，且在范围 [0, 231 - 1] 内
<br>题目数据保证答案是一个 32-bit 整数
