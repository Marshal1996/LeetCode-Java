## 题目描述

实现一个基本的计算器来计算一个简单的字符串表达式 s 的值。

 

示例 1：
<br>输入：s = "1 + 1"
<br>输出：2

示例 2：
<br>输入：s = " 2-1 + 2 "
<br>输出：3

示例 3：
<br>输入：s = "(1+(4+5+2)-3)+(6+8)"
<br>输出：23

提示：
<br>1 <= s.length <= 3 * 105
<br>s 由数字、'+'、'-'、'('、')'、和 ' ' 组成
<br>s 表示一个有效的表达式
